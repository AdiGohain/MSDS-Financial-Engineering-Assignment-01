Importing Financial Time Series Data¶

Overview¶

I will be using ML classifiers, including tree-based ensemble boosting methods, to predict the direction of agricultural commodity futures--(Corn,Wheat,Soybeans,Cotton) (up or down) using a number of lagged price features. In particular, I will be looking at daily closing spot prices for West Texas Intermediate (ticker WTI) with lags of one to seven days, as well as features based on opening, closing, high, and low price points, and daily trading volume.

A model for predicting the direction of daily returns sets the stage for testing the predictive utility of additional features. The domain of potential features or leading indicators is wide, including those associated with other price series, economic indicators, international events, securities filings, analyst and news reports, and media measures.

I am using financial time series data retrieved using the Python package yfinance, which interacts with the Yahoo! Finance application programming interface (API). This is a public-domain data showing historical prices for individual markets/companies, identified by ticker codes.

# List of commodities (CSV file names should match these keys)
commodities = ["Corn", "Soybeans", "Wheat", "Cotton"]

# Dictionary to hold DataFrames
dfs = {}

# Loop through each commodity and compute features
for commodity in commodities:
    file_path = f"{data_dir}/{commodity}.csv"
    
    # reading in the .csv
    df = pl.read_csv(file_path)
    
    # ensuring that the first column is Date
    if "Date" not in df.columns:
        df = df.rename({df.columns[0]: "Date"})
    
    # converting Date to datetime
    df = df.with_columns(pl.col("Date").str.slice(0, 10).str.strptime(pl.Date, "%Y-%m-%d"))
    
    # dropping useless columns Dividends and StockSplits
    drop_cols = [c for c in ["Dividends", "Stock Splits", "Adj Close"] if c in df.columns]
    df = df.drop(drop_cols)
    
    # creating lag price features
    df = df.with_columns(
        (pl.col("Close")).shift().alias("CloseLag1"),
        (pl.col("Close")).shift(2).alias("CloseLag2"),
        (pl.col("Close")).shift(3).alias("CloseLag3"))

        # creating high-minus-low (HML) for day and its lags
    df = df.with_columns(
        (pl.col("High") - pl.col("Low")).alias("HML"),
        (pl.col("High") - pl.col("Low")).shift(1).alias("HMLLag1"),
        (pl.col("High") - pl.col("Low")).shift(2).alias("HMLLag2"),
        (pl.col("High") - pl.col("Low")).shift(3).alias("HMLLag3"))
    
    # creating a net change for the day as the open minus closing price OMC
    # also creating the corresponding lag metrics
    df = df.with_columns(
        (pl.col("Open") - pl.col("Close")).alias("OMC"),
        (pl.col("Open") - pl.col("Close")).shift(1).alias("OMCLag1"),
        (pl.col("Open") - pl.col("Close")).shift(2).alias("OMCLag2"),
        (pl.col("Open") - pl.col("Close")).shift(3).alias("OMCLag3"))
    
    # creating volume lag metrics
    df = df.with_columns(
        pl.col("Volume").shift(1).alias("VolumeLag1"),
        pl.col("Volume").shift(2).alias("VolumeLag2"),
        pl.col("Volume").shift(3).alias("VolumeLag3"))
    
    # computing 10-day exponential moving averages of closing prices
    # computing acround CloseLag1 to avoid any "leakage" in explanatory variable set
    # note also the 10-day buffer between train and test in time-series cross-validation
    df = df.with_columns(
        pl.col("CloseLag1").ewm_mean(half_life=1, ignore_nulls=True).alias("CloseEMA2"),
        pl.col("CloseLag1").ewm_mean(half_life=2, ignore_nulls=True).alias("CloseEMA4"),
        pl.col("CloseLag1").ewm_mean(half_life=4, ignore_nulls=True).alias("CloseEMA8"))
    
    # log daily returns
    df = df.with_columns((np.log(pl.col("Close") / pl.col("CloseLag1"))).alias("LogReturn"))
        #(np.log(pl.col("Close") / pl.col("CloseLag1"))).alias("LogReturn"),
    df = df.with_columns(pl.when(pl.col("LogReturn") > 0).then(pl.lit(1)).otherwise(pl.lit(0)).alias("Target"))

    # Drop nulls (from initial lag rows)
    df = df.drop_nulls()

    
    # Save processed CSV
    dfs[commodity] = df
    out_file = f"{output_dir}/{commodity}-features.csv"
    df.write_csv(out_file)
    print(f"Processed {commodity}, saved to {out_file}")
